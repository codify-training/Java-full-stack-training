What is DOM (Document Object Model)?

The DOM (Document Object Model) is a programming interface for HTML documents. It represents the web page as a tree structure (like a family tree) where each element is a node/object.

In simple terms:
DOM = Your HTML page converted into objects that JavaScript can read and modify.

Why DOM is Important?

It allows JavaScript to:
Change HTML content
Change CSS styles
Add or remove elements
Handle events (click, input, change etc.)

Ex:
<!DOCTYPE html>
<html>
  <body>
    <h1 id="title">Hello World</h1>
    <p>This is a paragraph</p>
  </body>
</html>

DOM Tree (simplified):

Document
  └── html
       └── body
            ├── h1 (id="title")
            └── p

Note: We can see this kind of structure in our developer tools by clicking on inspect of the current window.

Example 1: Accessing an Element

// html file 
<h1 id="title">Hello World</h1>

// js file
const element = document.getElementById("title");

Explanation: Accessing element from html through id using getElementById method.

Example 2: Changing Content
// html file element
<h1 id="title">Hello World</h1>

// js file
const element = document.getElementById("title");
element.textContent = "hey! welcome back"
Explanation: when ever our javascript file runs then h1 element will be change from Hello world to hey! welcome back

<h1 id="title">Hello</h1>

el.style.color = "red";

Example-5: On Click event from html to JavaScript integration

// html file with button element
<button type="button" class="btn" onclick="buttonClicked()">click me</button>

// javascript file with function
function buttonClicked() {
    // block of code inside function
    alert("event clicked!");
}

Explanation:  Here, when ever button is clicked then buttonClicked() function will be called then block of code inside the function will be executed.

Example-6: Interacting HTML elements into JavaScript code using DOM

Step 1: Browser Loads HTML → Creates DOM

// id selector
<button id="btn">click me</button>

// class seelctor
<button class="btn">click me</button>

The browser converts this into a DOM tree (objects) like below

document
  └── body
        └── button (id="btn")

Step 2: JavaScript Accesses DOM

// getting element by id selector
const myButton = document.getElementById("btn");

// getting element by using class selector
const myButton = document.querySelector(".btn");


JavaScript:

Goes into DOM
Finds element with id="btn"
Stores it in myButton

Step 3: Event Listener process
myButton.addEventListener("click", function () {
    alert("Form Submitted");
})

Explanation:

This does NOT execute immediately.

Instead:
Registers a listener
Browser keeps watching for a click event

Step 4: User Clicks Button

When user clicks:
Browser detects event → sends it to JavaScript and then callback function executes and shows dialog box

final output:
alert("Form Submitted")

Life cycle flow:
1. HTML loads
2. DOM is created
3. JS runs
4. JS selects elements
5. JS registers event listener
6. User clicks button
7. Event triggers
8. Function executes
9. Alert is shown

1. Remove()
Removes an element from the DOM i.e. will remove element from web page

ex:
// html file
<p id="para">This will be removed</p>
<button onclick="removePara()">Remove</button>

// function js file
function removePara() {
  const element = document.getElementById("para");
  element.remove();
}

O/p:Paragraph disappears when button is clicked.

2.classList.add()
Adds a CSS class to an element. if styles is prensent in html or css file.

ex-1: styles which are present as internal CSS in html file

<style>
.sample {
  color: red;
  font-weight: bold;
}
</style>

<div id="heading" > hello world </div>


// js file
// creating a button element from java script
const btnElement = document.createElement("button");
btnElement.textContent = "remove";
document.body.appendChild(btnElement);

const headingEle = document.getElementById("heading");

btnElement.addEventListener("click",function(){
  // adding styles from html/css files with class selector
   headingElement.classList.add("sample");
})

O/P: if user clicks on button then styles will be added

3. classList.remove()
Removes a style class from an element.

btnElement.addEventListener("click",function(){
  // adding styles from html/css files with class selector
   headingElement.classList.remove("sample");
})

note: referring from the above example
O/P: if user clicks on button then styles will be removed for the html element

4.removeAttribute()
Removes an HTML attribute like inline styles, disabled etc...

Ex-1:
<div id="para" style="border: 1px solid black;margin: 10px;" >
    hello world
</div>
<button type="button" id="btn">click me</button>

const myPara = document.getElementById("para");
const myButton = document.getElementById("btn");


myButton.addEventListener("click", function () {
   myPara.removeAttribute("style");
})

ex-2:
<input type="text" id="inputBox" disabled>
<button onclick="enableInput()">Enable</button>

function enableInput() {
  const disabledEle =  document.getElementById("inputBox");
  disabledEle.removeAttribute("disabled");
}

Explanation: Here, remove attribute will be used to change the state of the input field.
Above example clicking on enableInput button input box will be enabled, user can enter data into it.


5. style.removeProperty()
Removes a specific CSS property.

ex-1:
<p id="para" style="color: red;">Hello</p>
<button onclick="removeStyle()">Remove Color</button>

function removeStyle() {
  const paraEle = document.getElementById("para");
  paraEle.style.removeProperty("color");
}

Explanation: Here, remove property method will remove particular style property of the element
