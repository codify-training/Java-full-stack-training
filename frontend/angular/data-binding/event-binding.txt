Event Binding:

=> event binding is a way to listen for and respond to user actions like clicks, keystrokes, or mouse movements. 
=> It allows data to flow from the View (HTML template) to the Component (TypeScript class). [1, 2, 3, 4] 

Core Concept
The basic syntax uses parentheses () around the event name and assigns it to a method defined in your TypeScript file. [5, 6] 

* Syntax: <element (event_name)="method_name()">
* Example: <button (click)="onSave()">Save</button> [1, 3, 6] 


Example: A Simple Button Click
This example shows how clicking a button in the HTML triggers a function in the code.

1. The Component (app.component.ts)
Define the function that should run when the event occurs. [1, 6] 

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})export class AppComponent {
  message: string = "";

  // This method is called when the button is clicked
  showMessage() {
    this.message = "Button was clicked! ";
  }
}

## 2. The Template (app.component.html)
Bind the button's click event to the showMessage() function.

<button (click)="showMessage()">Click Me!</button>
<p>{{ message }}</p>


## Advanced Usage: Accessing Event Data ($event) [9] 
Sometimes you need details about the event, such as which key was pressed or the value of an input field. You can pass the $event object to your method to access this information. [9, 10, 11] 
## Template Example

<!-- Passing the event object to the method -->
<input (input)="onInputChange($event)" placeholder="Type something...">


In your TypeScript, the $event object contains properties like target.value. [9, 12] 

onInputChange(event: any) {
  console.log("User typed:", event.target.value);
}

Common Event Binding Types
You can bind to any standard DOM event: [10, 13] 

* (click): Responds to mouse clicks.
* (input): Fires whenever the value of an <input> or <textarea> changes.
* (keyup): Detects when a keyboard key is released.
* (submit): Used with forms to handle submission logic. [1, 9, 11, 13, 14, 15, 16]
